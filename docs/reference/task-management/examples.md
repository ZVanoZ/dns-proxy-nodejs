## Пример 1: "Добавить поддержку протокола IPv6"

**Тип задачи:** Новая функциональность  
**Уровень сложности:** Level 3 (Intermediate Feature)  
**Требует творческой фазы:** Да

### Шаг 1: Инициализация

```
/van Добавить поддержку протокола IPv6 в DNS-прокси сервер. Необходимо обрабатывать AAAA записи и поддерживать IPv6 адреса в конфигурации.
```

**Результат:**
- Задача зарегистрирована в `memory-bank/tasks.md`
- Определен уровень сложности: Level 3
- AI готов к планированию

### Шаг 2: Планирование

```
/plan
```

**Результат:**
AI создаст план с этапами:
- Анализ текущей обработки DNS-запросов
- Добавление поддержки AAAA записей
- Обновление конфигурации для IPv6
- Тестирование IPv6 функциональности

### Шаг 3: Творческая фаза

```
/creative
```

**Результат:**
AI исследует варианты:
- Как обрабатывать IPv6 адреса в конфигурации
- Как интегрировать AAAA записи в существующую логику
- Как тестировать IPv6 функциональность

Создаются документы в `memory-bank/creative/creative-ipv6.md`

### Шаг 4: Реализация

```
/build
```

**Результат:**
AI реализует изменения согласно плану и творческим решениям, пишет тесты.

### Шаг 5: Рефлексия

```
/reflect
```

**Результат:**
AI анализирует реализацию, документирует уроки в `memory-bank/reflection/reflection-ipv6-[date].md`

### Шаг 6: Архивирование

```
/archive
```

**Результат:**
AI создает архивный документ `memory-bank/archive/archive-ipv6-[date].md`, помечает задачу как завершенную.
