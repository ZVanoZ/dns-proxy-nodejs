Внеси в memory-bank набор правил для автоматического журналирования чатов Cursor AI.
Используй методологию "vanzan01/cursor-memory-bank" v0.8.
Зарегистрируй это действие как отдельную задачу.

Правила:
1. При каждом создании чата, создавать пустой файл.
Путь к файлу по маске "./docs/prompt/<YYYY>/<YYYY-MM-DD>/<HH24-MM>/chat.md"
Путь формируется на основании момента времени по Гринвичу.
2. При старте и завершении каждого запрса выполнять экспорт чата в файл с заменой содержимого.
3. Для каждого запроса писать режим работы чата.
