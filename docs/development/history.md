# История разработки проекта

## Консольные команды, использованные при создании проекта

Этот документ содержит исторические команды, которые использовались при первоначальной настройке проекта. Они могут быть полезны для понимания процесса создания проекта с нуля.

### Инициализация TypeScript проекта

```bash
# Инициализация проекта
npm init -y

# Установка основных зависимостей
npm install dns-packet ini

# Установка зависимостей для разработки
npm install --save-dev typescript @types/node @types/dns-packet @types/ini ts-node

# Создание конфигурации TypeScript
npx tsc --init
```

### Настройка TypeScript

После выполнения `npx tsc --init` необходимо настроить `tsconfig.json` для работы с ESM модулями и строгой типизацией.
