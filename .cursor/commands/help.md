# Команда "help"

Эта команда для получения помощи по использованию программы Cursor.

## Алгоритм работы

1. Определить версию программы Cursor, которая установлена у пользователя.
2. Определить версию программы Cursor, по которой задан вопрос, если версия указана.
3. Определить актуальную стабильную версию программы курсор.
4. Попытаться найти ответ для версии, которая установлена у пользователя.
5. Если в вопросе присутствует версия Cursor, то найти ответ для нее и сравнить с версией, которая установлена у пользователя.
6. Если актуальная версия Cursor не соответствует текущей, то попытаться найти ответ для актуальной версии Cursor и выдать рекомендации по миграции.
7. Выдать сравнительную таблицу по версиям с краткой аннотацией результата анализа предыдущих пунктов алгоритма.
