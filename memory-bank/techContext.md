# Memory Bank: Technical Context

## Технологический стек

### Основные технологии
- **TypeScript 5.9.3** - строгая типизация, современный синтаксис
- **Node.js** - ESM модули (nodenext), поддержка Node 20+
- **ESBuild 0.24.0** - быстрая сборка проекта

### Зависимости
#### Production
- `dns-packet ^5.6.1` - работа с DNS пакетами
- `ini ^6.0.0` - парсинг INI конфигураций
- `pino ^10.1.0` - структурированное логирование
- `pino-pretty ^13.1.3` - форматирование логов для разработки

#### Development
- `jest ^29.7.0` - фреймворк тестирования
- `ts-jest ^29.1.2` - поддержка TypeScript в Jest
- `tsx ^4.21.0` - выполнение TypeScript без компиляции
- `typescript ^5.9.3` - компилятор TypeScript

### Конфигурация TypeScript
- **Module System**: nodenext (ESM)
- **Target**: esnext
- **Strict Mode**: включен
- **Source Maps**: включены
- **Declaration Maps**: включены

### Архитектурные особенности
- **ESM модули** - использование import/export
- **Строгая типизация** - strict mode, noUncheckedIndexedAccess
- **Модульная структура** - разделение на App, Logger, Test модули
- **Конфигурация через INI** - гибкая настройка через файлы

### Сборка и развертывание
- **Сборка**: скрипты в `scripts/` для очистки, сборки зависимостей и приложения
- **Docker**: поддержка DEV и PROD профилей
- **Конфигурация**: настройка через `config/app.ini`

### Логирование
- Ротация логов по дням
- Структурированное логирование через Pino
- Форматирование для разработки через pino-pretty
