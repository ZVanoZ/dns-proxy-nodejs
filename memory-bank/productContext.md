# Memory Bank: Product Context

## Продукт: dns-proxy-nodejs

### Назначение
DNS-прокси для:
1. Разработки сетевых приложений
2. Маршрутизации DNS-запросов при работе через VPN

### Целевая аудитория
- Разработчики сетевых приложений
- Пользователи, работающие через VPN
- Системные администраторы

### Основные функции
- Проксирование DNS-запросов
- Настраиваемая конфигурация
- Логирование операций
- Поддержка различных режимов работы

### Режимы работы
1. **DEV режим** - разработка с отладкой
2. **PROD режим** - продакшен скомпилированного приложения
3. **Docker режим** - контейнеризированное развертывание

### Требования к окружению
- Node.js (версия не указана, но используется ESM)
- Доступ к порту 53 (для стандартного DNS) или 5053 (для нестандартного)
- Конфигурационный файл `config/app.ini`

### Интеграция
- Работа с системными DNS-запросами
- Поддержка iptables для перенаправления портов
- Docker контейнеризация для изоляции

### Мониторинг
- Структурированное логирование
- Ротация логов по дням
- Возможность отладки через debugger
