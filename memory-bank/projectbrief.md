# Memory Bank: Project Brief

## Проект: dns-proxy-nodejs

### Описание
DNS-proxy-nodejs - небольшой DNS-прокси для использования при разработке сетевых приложений или для маршрутизации DNS-запросов при работе через VPN.

### Текущее состояние
**Альфа-версия.** Функционал работает. Выполняется рефакторинг и тестирование.

### Основные возможности
- DNS-прокси сервер на Node.js
- Поддержка UDP протокола
- Настраиваемая конфигурация через INI файлы
- Ротация логов
- Поддержка Docker контейнеризации
- Режимы разработки (DEV) и продакшена (PROD)

### Технологический стек
- **Язык**: TypeScript
- **Платформа**: Node.js (ESM модули)
- **Зависимости**: dns-packet, ini, pino (логирование)
- **Сборка**: esbuild
- **Тестирование**: Jest
- **Контейнеризация**: Docker, Docker Compose

### Структура проекта
- `src/` - исходный код TypeScript
- `dist/` - скомпилированный код
- `config/` - конфигурационные файлы
- `scripts/` - скрипты сборки и запуска
- `tests/` - тесты
- `docs/` - документация

### Цели проекта
1. Предоставить простой DNS-прокси для разработки
2. Обеспечить гибкую настройку через конфигурационные файлы
3. Поддержать различные режимы работы (DEV/PROD)
4. Обеспечить качественное логирование и мониторинг
